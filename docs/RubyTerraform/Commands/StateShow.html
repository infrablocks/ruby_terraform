Overview

Wraps the terraform state show command which shows the attributes of a resource in the Terraform state.

This command shows the attributes of a single resource in the Terraform state. The :address argument must be used to specify a single resource. You can view the list of available resources with StateList.

For options accepted on construction, see Base#initialize.

When executing an instance of StateShow via Base#execute, the following options are supported:

  • :address: the module address or absolute resource address of the resource instance to show; required.

  • :chdir: the path of a working directory to switch to before executing the given subcommand.

Examples:

RubyTerraform::Commands::StateShow.new.execute(
